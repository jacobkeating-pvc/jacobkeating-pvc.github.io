Operating Systems Terminology

Term

Definition
Operating Systems The Software that controls the overall operation of a computer. It provides the means by which a user can store and retrever files, provides the interface by which a user can request the executions of programs and provides the environment necessary to execute the programs requested.
Job Queue The queue of program excecutions in mass storage waiting to be executed.
FIFO First-in, first-out. That is the objects are removed from the queue in the order in which they arrived
Interactive Processing Operating Systems developed that allowed a program being executed to carry on a dialogue with the user through remote terminals.
Time Sharing This is when the computer is forced to execute tasks under a deadline.
Multiprogramming This is a process by which time is divided into intervals and then the execution of each job is restrected to only one interval at a time.
Application Software Application software consists of the programs ofr performing tasks particular to the machine's utilization.
System Software System software performs tasks that are common to computer systems in general.
Utility Software Utility software extends the capabilities of the operating system.
Graphical User Interface (GUI) Communicates with the user by graphically illustrating objects to be manipulated, such as files and programs, are represented pictorally on the display as icons. These systems allow the user to issue commands by using on of the several common input devices.
Shell Older method of communication between the operating system and the user, usually through textual messages using a keyboard and monitor screen.
Kernel The internal part of the operating system containing the software components that perform the very basic functions required by the computer installation.
Device Driver The process by which the software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
File Manager whose job is to coordinate the use of the machine's mass storage facilities. More precisely the file manager records of all the files stored in mass storage
Memory Manager The process by which the system is charged with the task of coordination the machine's use of main memory
Virtual Memory "Fictional Memory Space" created by paging.
Boot Strapping The process by which the operating system is transfered from mass storage into main memory.
ROM Read Only Memory. Memory allocation in which its contents can only be read and not altered
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resouce that is allocated to another.
Spooling This is a techniqe of holding data for output at a later but more convenient time.
Super User (or administrator) Accounts with highly privileged access to the operation system by identifing the individual as the admin.
Multi-Factor Authentication Multi-process authentication, is a security method that requires users to provide two or more different forms of identification before granting access to a system or account, enhancing protection against unauthorized access.
