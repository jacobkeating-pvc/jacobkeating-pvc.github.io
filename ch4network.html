Operating Systems Terminology

Term

Definition

Network Classifications
MAN Metropolitan Area Network normally used for networks of intermediate size.
LAN Local Area Network links machines in a single building or campus.
WAN Wide Are Network links machines over a long distance.

Network Models
Bus Network A method of connecting various machines through a common bus line.
Star Network A method of connecting various machines by each linking to a single network switch.

Protocols
CSMA/CD Ethernet Carrier Sense, Multiple Access with Collision Avoidence Ethernet dictates that each message be brocasted to all the machines on the bus connected by Ethernet.
CSMA/CA Wifi Carrier Sense, Multiple Access with Collision Avoidence Wifi much like Ethernet connects multiple machines but rather than trying to eliminate collisions it avoids them from the beginning.

Combining Networks
Repeater A device that passes signals back and forth between the two origional buses.
Bridge A simmilar but more complex repeater which connectes two machines directly by looking at the destination address that accompanies each message.
Switch Essentially a bridge with multiple connections, allowing it to connect to several busses.
Router Special purpose computers designed to forward messages over WiFi.
Gateway Acts as the connection between a network and the internet.

Client/Server Model
P2P: peer-to-peer An interprocess communication where the users service and recive eachothers data.
Distributed System A interaction between computers consisting of software units that execute as processes on different computers.
Cloud Computing A method of connecting huge pools of shared computers on the network and allocates them for use by the client.

Internet Addressing
Tier 1 The topmost layer ISPs whereby highspeed, high-capacity, international WANs opperate.
Tier 2 The middle layer ISPs network focused morso on a regional scope and at a slower bandwidth.
Tier 3 Access layer ISP focused on connecting to all the end systems of a service.

IP Address
ICANN
DNS Domain Name Services are name servers used as an Internet-wide directory system.
TLD: Top-Level Domain These are special domains marked by .edu .gov etc...
Country-Code TLDs A TLD where two letter distinctions are made between regions.

Internet Protocols
FTP File Transfer Protocal, an internet application used for accessing another computer from a great distance. It was later renamed to SSH.
HTTP Hyper Text Transfer Protocol used for sending websites data to clients.
SMTP Simple Mail Transfer Protocol defines a way that two computers on the network may interact when trasmitting an email message from one host to the other.

World Wide Web
URL Uniform Resource Locator designed to locate and retrieve documets on the Web, each document by assigning a unique address to each.
Client-Side Activities These are actions performed by a client such as a browser.
Server-Side Activities These are actions performed by a server such as a webserver.

Internet Four-Layer Software
Application Software The application layer consists of those software units such as clients and servers that use Internet communication to carry out their tasks.
Transport Layer The transport layer accepts messages from the application layer ensures that the messages are properly formatted for transmition over the Internet.
Network Layer The network layer decides which direction a packed of information should be sent at each step along the packet's path across the Internet.
Link Layer The link layer has the responsibility of transferring the packet.

Internet is:

Cybersecurity
Malware Malicious software transferred to, and executed on, the computer itself.
Virus A piece of software that infects a computer by inserting itself it programms that already reside in the machine.
Worm A piece of automated software that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
Trojan Horse A program that enters a computer system disguised as a desirable program, such as a game or useful utility package.
Spyware Software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
Phishing A technique of obtaining information explicitly by simply asking for it.
DDoS Attack The process of overloading a computer with messages to overwhelm the targeted server.

Protection
Firewall A primary prevention techniqe to filter traffic passing through a point in the network.
Spam Filter Filter designed to block unwanted email and messages
Proxy Server A software unit that acts as an intermediary between a client and a server.
Antivirus Software A software used to detect and remove the presence of known viruses and other infections.
Cryptography The study of sending and receiving messages securely in the presence of adversaries.
Public Key Encryption Encryption that can be used to transform a message into a coded version such that even if the encrypted message falls into unscrupulus hands, the encoded information will remain confidential.
