Programming Languages

Term

Definition
Machine Language A programming language consisting of ones and zeros.
Debugging The process of locating errors in code and correcting them.
Assembler Assemblers convert mnemonic expressions into machine language instructions.
Assembly Language A mnemonic system for representing programs collectively.
Translator Translates programs expressed in high-level primitives into machine-language programs.
Compiler A translator that converts high-level primirives into machine code.
Interpereter Interpereters are simmilar to translators except they execute the instructions as they translate.
Cross-Platform Software Software that functions across different operating systems.
Natural Language A language spoken in the common vernacular of people, defined by its precice grammars.
Formal Lanugage A language such as a programming language.
Machine Independance A programming language that does not rely on the characteristics of a particular machine.
High-Level Programming Language Programming languages that are closer to natural languages from formal languages.
Object-Oriented Programming (OOP) Programming, where software is viewed as a collection of units called objects, each of which is capable of performing actions that are immediately related to itself as well as requesting actions of other objects.
Scripting Language A programming language that is used to manipulate, customize, and automate the facilities of an existing system.


Data Types
Integer Numeric data consisting of whole numbers.
Float Numeric data that might contain values other than whole numbers.
String A datatype consisting of characters.


Primitive Data Type Data types that are included as primitives in a programming language.
Data Structure The conceptual shape or arrangement of data.


Python Collection Data Structures
List Lists are used to store multiple items in a single variable.
Tuple A tuple is a collection which is ordered and unchangeable.
Set A set is a collection which is unordered, unchangeable*, and unindexed.
Dictionary Dictionaries are used to store data values in key:value pairs. A dictionary is a collection which is ordered*, changeable and do not allow duplicates. Dictionaries are written with curly brackets, and have keys and values.


Function A set of instructions for performing a task tha can be used as an abstract tool by other program units.
Local Variable A variable declaired within a function.
Global Variable A variable whose scopes are not restricted to a single part.
Source Program A program in its original form, as written by the programmer.
Object Program Object Programs are programs in machine language form.
